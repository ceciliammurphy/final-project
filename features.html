Trulytics Features

Trulytics is designed to present social media analytics in a clearer and more useful way. Instead of only showing surface-level numbers, the prototype combines exported Instagram data with manually collected interaction metrics to highlight patterns, comparisons, and context that help users better understand their content and audience.

Follower Growth

The follower growth feature tracks how an account's audience changes over time using exported Instagram data. Instead of only showing a single follower count, this view highlights whether growth is steady, inconsistent, or influenced by specific time periods. By visualizing this data, Trulytics helps users better understand how their audience evolves.

View This Feature

Following Growth

This feature tracks how the number of accounts followed changes over time. When compared with follower growth, it provides additional context about account behavior and patterns. This comparison helps users better understand their relationship to their audience rather than viewing follower counts in isolation.

View This Feature

Post Engagement (Interactions)

This feature visualizes post engagement using manually collected interaction data, including likes, comments, and sends. Instead of focusing on a single metric, it provides a more complete view of how audiences respond to content. By comparing interactions across posts, users can more easily identify patterns in performance.

View This Feature

Posting Activity

Posting activity is based on timestamps from the exported posts data. This feature shows how often content is published over time, helping users connect their posting behavior with changes in audience growth and engagement patterns.

View This Feature

Content Pattern Breakdown

This feature organizes content by type, such as carousel posts, single images, and video. By grouping content in this way, Trulytics highlights patterns in how different formats are used and how they may contribute to visibility and engagement strategies.

View This Feature

Why These Features Matter

Together, these features reflect the larger purpose of Trulytics: helping users understand social media performance in a way that feels more complete, transparent, and useful. By combining real data with clearer visual organization, the dashboard makes it easier to interpret patterns in growth, behavior, and engagement rather than relying on isolated metrics.
