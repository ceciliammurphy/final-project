Methods

This project explores how social media analytics shape the way users interpret engagement, visibility, and performance online. The goal of Trulytics is to demonstrate how clearer data presentation could help users better understand patterns in their content and audience behavior. To create this prototype, I combined exported Instagram data with web development tools to simulate how a more transparent analytics platform might function.

Data Collection

To build the prototype, I exported my personal Instagram data through Meta’s data download tool. I selected categories related to engagement and interaction patterns, including likes, comments, media, story interactions, and follower information. These data categories provide insight into how audiences respond to posts and how engagement changes over time.

The exported data serves as an example dataset that demonstrates how analytics tools organize and interpret user activity. Rather than focusing on exact numbers, the project emphasizes how data structure and visualization influence how users understand performance.

Data Preparation

After exporting the data, I reviewed the files to identify which categories were most relevant for demonstrating engagement trends and content performance. Because social media data often appears in complex formats, I simplified the information into smaller datasets that could be used to simulate analytics visuals. The goal was not to replicate Instagram’s exact system, but to demonstrate how analytics could be presented more clearly and meaningfully.

Sample data points were organized into categories such as follower growth, engagement rates, and content comparison metrics. These categories informed the structure of the dashboard prototype.

Prototype Development

[Prototype Screenshot: Dashboard Layout]

The Trulytics prototype was created using HTML, CSS, and JavaScript in Visual Studio Code. These tools were used to simulate how analytics dashboards visually present information through charts, layout structure, and interactive components. JavaScript allows numerical data to be represented visually, while CSS controls the layout and design of the interface.

The prototype focuses on readability and organization. Each feature is presented as a separate data module, allowing users to interpret metrics individually while still understanding broader patterns. The dashboard layout was designed to resemble modern analytics tools while remaining simple enough to demonstrate the concept clearly.

Design Decisions

The visual design of Trulytics was influenced by existing social media interfaces, particularly Instagram’s color palette and minimal layout style. The goal was to create a familiar environment that reflects how users already experience social media platforms. The centered layout, card-based sections, and consistent spacing help guide attention toward the data itself.

Design choices prioritize clarity over decoration. By reducing visual clutter, the prototype demonstrates how analytics can be presented in a way that feels more understandable and accessible. The design also supports the broader argument of the project: that the way data is presented affects how users interpret online performance.

Limitations

Because Trulytics is a prototype, the data used in the dashboard is simplified and manually structured. The project does not connect directly to Instagram’s live API, meaning the analytics are not updated in real time. Instead, the prototype demonstrates how a platform could present analytics differently if given access to user data.

The purpose of the project is not to replace existing analytics systems, but to explore how alternative design approaches might influence how users interpret engagement and performance metrics.

Why Methods Matter

Understanding how data is collected, structured, and visualized helps reveal how social media platforms shape user perception. Analytics are not neutral; they influence how individuals evaluate success, visibility, and audience response. By experimenting with alternative forms of data presentation, Trulytics demonstrates how design and analytics work together to influence interpretation.
